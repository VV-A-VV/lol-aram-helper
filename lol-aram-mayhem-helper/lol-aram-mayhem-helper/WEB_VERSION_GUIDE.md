# 🌐 网页版使用说明

## 快速开始

1. **打开文件**：双击 `web-version.html` 即可在浏览器中打开
2. **输入英雄名称**：在搜索框中输入英雄名称（如：亚索、劫、阿狸）
3. **点击查询**：自动生成 op.gg 链接
4. **查看详情**：点击按钮跳转到 op.gg 查看完整数据

## 特点

✅ **无需安装** - 直接在浏览器中运行
✅ **跨平台** - Windows/macOS/Linux 都可用
✅ **简单易用** - 输入英雄名即可
✅ **美观界面** - 现代化设计

## 支持的英雄

目前支持常见英雄的中文名称，包括：
- 亚索、劫、阿狸、盖伦
- 拉克丝、金克斯、卡特琳娜
- 提莫、薇恩、德莱文
- 伊泽瑞尔、奎因、锐雯
- 等等...

也可以直接输入英文名称（如：yasuo、zed）
